## How to use the calendar
1. Download code.html
2. <a href="calendar.google.com">Open Google Calendar</a> settings and make your calendar public
3. Enter the URL in line.156
4. Save it and display it in your browser. If the calendar is displayed, it's OK.

## Weather API Settings
We use <a href="https://open-meteo.com/">Open-Meteo</a> to display the weather.

1. First, go to <a href="https://open-meteo.com/">open-meteo</a>.
2. Click "Try the API" here
3. Set your coordinates and TZ, and the hourly weather variables below.(It might be quicker to use the search on the right lol)
※In reality, you will need to modify the JS code, but this is something that you should control.
